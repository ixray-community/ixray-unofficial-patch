# IX-Ray Unofficial Patch CoP

## Overview

It is based on [Unofficial Patch CoP 1.6.03](https://www.amk-team.ru/forum/topic/12644-unofficial-patch-cop-1603/) by __Jurok__. You can use this addon as a base for your modifications, because there are many bugs fixed here

The main goal of the __Unofficial Patch__ is to fix all errors and bugs of the original game and bring the game to its highest quality, while preserving the original balance and graphics as much as possible

## Requirements

Installation:

1. Install original Stalker Call of Pripyat 1.6.02
2. Install current version of IX-Ray
3. In root folder of the game create a folder `ixr_addons`
4. Extract addon archive into the `ixr_addons` folder
5. Start the game and play

> [!IMPORTANT]
> New game save required. The game will crash if you try to load a save without the mod loaded.

Link to the current __IX-Ray__ [release](https://github.com/ixray-team/ixray-1.6-stcop/releases/latest)
